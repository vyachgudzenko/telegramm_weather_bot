# telegramm_weather_bot
Telegram bot that reports the weather for today. To do this, use the information from the site sinoptik.ua
The configuration file contains a variable token. Your token for controlling the bot is indicated there.
During operation, the bot receives only one request - the name of the city in Russian and shows the weather for today.
If incorrect data is entered, a message is sent asking you to try again

В файле настройки находится переменная токен. Там указывается Ваш токен для управления ботом. 
ВО время работы, бот получает только один запрос - название города на русском языке и показывает погоду на сегодня. 
В случае, если введенны некорректные данные - присылается сообщение с просьбой повторить попытку
